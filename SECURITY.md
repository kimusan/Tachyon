# Security Policy

## Supported Versions

Only the latest release receives security updates.

| Version  | Supported |
| -------- | --------- |
| 3.x      | ✔         |
| < 3.0.0  | ❌         |

## Reporting a Vulnerability

Please report security issues as an encrypted email. Your report should include clear steps to reproduce and a classification of the vulnerability.

Reports will be analyzed and fixed as fast as possible. Disclosure is planned together with the reporter. Credits are granted and can be included in all public communication if desired.

## Upgrade path

Existing SnappyMail installations (2.x) can upgrade directly to Tachyon 3.x. The on-disk data format is backward compatible. PHP 8.2+ is required.
